<h1 align="center">3DGS-Viewer</h1>
It is a simple 3DGS viewer based on Viser

## Run
To use the viewer, you can use the following command:
```
python webui.py --gs_source "<YOUR PATH>" --colmap_dir "<YOUR PATH>"
```
